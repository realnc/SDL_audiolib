Detailed Description

libFLAC decoder.

The decoder supports raw FLAC data (as found in .flac files) or FLAC streams inside an Ogg container (as found in .ogg or .oga files.) The format can be automatically detected if the input is seekable.

Note
For Ogg support to work, libFLAC must have been built with Ogg support.
